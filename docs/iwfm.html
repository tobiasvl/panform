Inweb-flavoured Markdown reader for Pandoc

  • §1. Inweb-flavoured Markdown

§1. Inweb-flavoured Markdown.The [foundation](https://ganelson.github.io/inweb/foundation-module/#5) library for [Inweb](https://ganelson.github.io/inweb/) defines a custom Markdown variant called [Inweb-flavoured Markdown](https://ganelson.github.io/inweb/foundation-module/5-mv.html) (IWFM), which is used to write Inweb webs.

This Pandoc reader parses IWFM.
